Bienvenido a Angular Dashboard

¿Quién Soy?

Soy un desarrollador apasionado por crear aplicaciones web modernas y funcionales. Este proyecto es un ejemplo de una aplicación full-stack que combina un backend seguro con autenticación JWT y un frontend dinámico construido con Angular.

¿A qué me dedico?

Me dedico a construir soluciones que integran autenticación, autorización y gestión de usuarios. Este proyecto incluye un sistema de login, registro y protección de rutas, diseñado para demostrar cómo implementar estas funcionalidades en un entorno real.
